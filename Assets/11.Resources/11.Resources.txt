# Resources
- Resources 폴더에 저장된 에셋을 런타임에 동적으로 로드하거나 언로드할 수 있게 해주는 기능입니다.

# Static Methods
	메서드								설명																사용 설명
- Load<T>(string path)					Resources 폴더에서 특정 결로의 에셋을 동기적으로 로드합니다.		폴더 내의 에셋을 로드할때
- LoadAll<T>(string path)				해당 경로에 있는 모든 타입 T의 에셋을 배열로 반환합니다.				폴더 내의 여러 에셋을 한꺼번에 로드할때
- LoadAsync<T>(string path)				에셋을 비동기로 로드합니다. 로딩 중 다른 작업 수행 가능				로딩 화면 등에서 느료지는 걸 방지하고 싶을때
- UnloadAsset(obejct assetToUnload)		명시적으로 에셋을 메모리에 언로드합니다.							사용 중인 에셋을 언로드하면 오류 발생
- UnloadUnusedAssets()					사용하지 않는 에셋들을 메모리에서 자동으로 정리합니다.				씬 전환 후, 메모리를 줄이고 싶을 때 주로 사용

## 주의 사항
- Resources.Load는 런타임 메모리를 증가시킬 수 있으므로 남용을 피해야 합니다.
- 대규모 프로젝트에서는 Addressables 사용 권장(동적 에셋 관리에 더 유리함)


## 동기와 비동기 
항목					Load								LoadAsync
동기/비동기			동기									비동기
게임멈춤여부			로딩 중 잠깐 멈춤 가능				멈추지 않음
사용 편의성			간단									코루틴 사용
리소스 크기			작거나, 즉시 필요한 리소스			큰 리소스 또는 로딩 분산에 적합
대표 사용 사항		씬 초기화, 튜토리얼, 빠른 테스트		로딩화면, 스테이지 전환, UI팝업
