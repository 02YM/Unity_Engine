# RaycastHit
- 광선이 어떤 물체와 충돌했을 때 해당 지점에 대한 정보를 담고 있는 구조체입니다.
- 광선이 어떤 지점에 맞았는지, 어떤 객체인지, 어디를 맞았는지 알 수 있게 해줍니다.

# Properties
	프로퍼티					설명									활용예시
- collider				광선이 충돌한 콜라이더				어떤 오브젝트를 맞췄는지 확인할 때 사용
- point					충돌 지점의 월드 좌표					이펙트 생성, 피격 위치 표시
- normal				충돌 지점의 표면 방향					반사 벡터 계산, 바닥 감지, 슬라이딩 등
--	맞은 오브젝트의 표면 방향입니다, 오브젝트 기준입니다.
- distance				광선 시작점과 충돌 지점 사이의 거리	시야 거리 제한, 타겟과의 거리 측정
- transform				충돌한 오브젝트의 Transform			위치나 회전 정보 접근 시 사용


## hit.normal
- 충돌한 표면의 수직 방향을 나타는 벡터
-- 단위 벡터
-- 표면이 향하고 있는 방향을 나타냄
-- 길이는 항상 1
-- 충돌 표면에서 반사, 이펙트 정렬, 기울기 계산 등에 사용됨

## 사용 예시
	상황							활용방법
총알 이펙트를 벽테 붙일때			벽의 방향으로 이펙트를 정렬하기 위해 사용
반사 벡터 계산(레이저 반사)		Vector3.Reflect 함수에 인자로 사용
AI가 경사면 위에 있는지 판단		hit.normal 과 Vector3.up을 비교
물리 슬라이딩 구현				표면 방향을 따라 미끄러지는 방향 계산
